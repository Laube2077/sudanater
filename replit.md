# Sudanater (スダネイター！)

## Overview
A simple interactive website with HTML, CSS, and JavaScript.

## Project Structure
- `index.html` - Main HTML file
- `style.css` - Styling with gradient background and animations
- `script.js` - Interactive button click counter

## Running the Project
The site is served using Python's built-in HTTP server on port 5000:
```
python3 -m http.server 5000 --bind 0.0.0.0
```

## Features
- Beautiful purple gradient background
- Interactive click counter button
- Smooth animations and transitions
- Responsive design

## Deployment
Configured for autoscale deployment using Python HTTP server.
